## Sistema RST
El sistema RST se utiliza para dar informes de señal.
- **R** (Readability): Legibilidad (1 a 5).
- **S** (Signal Strength): Fuerza de la señal (1 a 9).
- **T** (Tone): Tono (1 a 9, solo para CW).

Un informe típico sería "59" (cinco-nueve) para una señal perfectamente legible y muy fuerte.
