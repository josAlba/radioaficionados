# Código Q

El Código Q es un conjunto de abreviaturas de tres letras que se utiliza en radiocomunicaciones para agilizar la transmisión y superar barreras idiomáticas.

| Código | Significado (Pregunta / Afirmación) |
|--------|-----------------------------------|
| **QRG**| ¿Me indica mi frecuencia exacta? / Su frecuencia exacta es... |
| **QRL**| ¿Está la frecuencia ocupada? / La frecuencia está ocupada. |
| **QRM**| ¿Tiene interferencias? / Tengo interferencias (1 a 5). |
| **QRN**| ¿Le molestan los ruidos atmosféricos? / Tengo ruidos atmosféricos (1 a 5). |
| **QRP**| ¿Debo disminuir la potencia? / Disminuya la potencia. |
| **QRQ**| ¿Debo transmitir más rápido? / Transmita más rápido. |
| **QRS**| ¿Debo transmitir más despacio? / Transmita más despacio. |
| **QRT**| ¿Debo dejar de transmitir? / Deje de transmitir. |
| **QRV**| ¿Está usted preparado? / Estoy preparado. |
| **QRX**| ¿Cuándo me llamará de nuevo? / Le llamaré a las... |
| **QRZ**| ¿Quién me llama? / Le llama... |
| **QSB**| ¿Tengo desvanecimiento de señal (fading)? / Tiene fading. |
| **QSL**| ¿Puede darme acuse de recibo? / Le doy acuse de recibo. |
| **QSO**| ¿Puede comunicar con...? / Puedo comunicar con... |
| **QSY**| ¿Debo cambiar de frecuencia? / Cambie a la frecuencia... |
| **QTH**| ¿Cuál es su ubicación? / Mi ubicación es... |
| **QTR**| ¿Cuál es la hora exacta? / La hora exacta es... |
