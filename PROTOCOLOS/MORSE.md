# Código Morse (CW)

El código Morse o telegrafía es un sistema de representación de letras y números mediante señales emitidas de forma intermitente.

## Alfabeto Morse

| Letra | Código | Letra | Código |
|-------|--------|-------|--------|
| **A** | `.-`   | **N** | `-.`   |
| **B** | `-...` | **O** | `---`   |
| **C** | `-.-.` | **P** | `.--.` |
| **D** | `-..`  | **Q** | `--.-` |
| **E** | `.`    | **R** | `.-.`  |
| **F** | `..-.` | **S** | `...`  |
| **G** | `--.`  | **T** | `-`    |
| **H** | `....` | **U** | `..-`  |
| **I** | `..`   | **V** | `...-` |
| **J** | `.---` | **W** | `.--`  |
| **K** | `-.-`  | **X** | `-..-` |
| **L** | `.-..` | **Y** | `-.--` |
| **M** | `--`   | **Z** | `--..` |

## Números

| Número | Código | Número | Código |
|--------|--------|--------|--------|
| **1**  | `.----`| **6**  | `-....`|
| **2**  | `..---`| **7**  | `--...`|
| **3**  | `...--`| **8**  | `---..`|
| **4**  | `....-`| **9**  | `----.`|
| **5**  | `.....`| **0**  | `-----`|

## Signos de Puntuación Comunes

| Signo | Código |
|-------|--------|
| **Punto (.)** | `.-.-.-` |
| **Coma (,)** | `--..--` |
| **Interrogación (?)** | `..--..` |
| **Barra (/)** | `-..-.` |

## Abreviaturas y Prosignos
- **CQ**: Llamada general (Calling Any Station).
- **DE**: De (usado antes del indicativo del emisor).
- **K**: Invitación a transmitir.
- **SK**: Fin de contacto (Silent Key).
- **73**: Saludos cordiales.
- **88**: Besos y abrazos.
